# 🩺 Diabetes Prediction System

An AI-powered web application that predicts whether a person is diabetic based on clinical parameters.

---

## 🚀 Features

- Machine Learning based prediction  
- Real-time analysis  
- Interactive UI using Streamlit  
- Multiple model comparison  

---

## 🧠 Model Used

- Random Forest / XGBoost (Best performing model selected)

---

## 📊 Input Parameters

- Pregnancies  
- Glucose  
- Blood Pressure  
- Skin Thickness  
- Insulin  
- BMI  
- Diabetes Pedigree Function  
- Age  

---

## 📈 Accuracy

- Achieved accuracy: **~75% – 82%**
- Improved using:
  - Feature preprocessing  
  - Stratified sampling  
  - Hyperparameter tuning  

---

## 🛠️ Tech Stack

- Python  
- Pandas, NumPy  
- Scikit-learn  
- XGBoost  
- Streamlit  

---
